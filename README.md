# Priyansh Kumar - Personal Portfolio 🚀

A sleek, modern, and highly interactive personal portfolio website designed to showcase my journey as a Data Science and Computer Science student. It features a dark-themed UI with amber accents, smooth scroll animations, and a responsive grid layout.

🔗 **[View Live Demo Here](#)** *(Replace with your deployed link, e.g., Vercel or Netlify)*

## ✨ Features
* **Modern UI/UX:** Dark theme with custom glowing ambient backgrounds and hover effects.
* **Smooth Animations:** Integrated with Framer Motion for staggered loading, floating icons, and scroll-triggered reveals.
* **Grid Layouts:** Custom card-based grid system for the Technical Skills section using Lucide React and custom SVG icons.
* **Fully Responsive:** Mobile-first approach ensuring the site looks perfect on phones, tablets, and desktops.
* **Smooth Scrolling:** Effortless navigation between the Hero, Education, Skills, Projects, and Contact sections.

## 🛠️ Tech Stack
* **Framework:** [React](https://reactjs.org/) (initialized with [Vite](https://vitejs.dev/))
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/) & Custom SVGs
